# Test Driven Development

Implement from [Kent Beck](https://en.wikipedia.org/wiki/Kent_Beck) theory

## Test Driven Development Cycle

1. Create test file

2. Run all testing

3. Write code

4. Run tests

5. Refactor code

- Repeat
